Uma página web moderna, elegante e totalmente responsiva desenvolvida para apresentar as principais novidades, sedes, mídias e a revolução no formato da **Copa do Mundo de 2026**.

---

## 📌 Sobre o Projeto

Este projeto foi reestruturado seguindo as melhores práticas de desenvolvimento web, separando a estrutura (*HTML*) da estilização (*CSS*). A página conta com uma interface imersiva inspirada na identidade visual do torneio, excelente escaneabilidade e elementos de mídia integrados.

### 🚀 Funcionalidades e Seções
* **Visão Geral:** Introdução ao contexto histórico e inclusão da imagem oficial do projeto (`copa.png`).
* **Três Países, Um Continente:** Exibição dinâmica em formato de *cards* das três nações anfitriãs (EUA, México e Canadá).
* **A Revolução no Formato:** Tabela comparativa detalhando as diferenças cruciais entre o formato antigo (até 2022) e o novo modelo de 48 seleções.
* **Assista aos Detalhes:** Seção de mídia com um player de vídeo do YouTube totalmente responsivo.
* **Fatos Marcantes:** Lista de curiosidades sobre logística regional, regras de classificação e recordes.
* **Design Responsivo:** Layout adaptável feito com CSS Grid e Flexbox para visualização perfeita em computadores e dispositivos móveis.
